# Exercicios-De-Logica-VisualG-PROA
Repositório com 140 exercícios de lógica de programação desenvolvidos em VisualG (PROA). Os exercícios abordam operações matemáticas, estruturas condicionais (se, senão), operadores lógicos (and, not) e laços de repetição (enquanto/Para), com foco no desenvolvimento do raciocínio lógico .
